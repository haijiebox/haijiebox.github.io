二分查找

标准二分查找

没有重复值

  • 循环条件：lo <= hi
  • 左边界更新：lo = mid + 1
  • 右边界更新：hi = mid - 1
  • 返回值 mid / -1

左边界二分查找

寻找有序数组中目标值第一次出现的位置（可能有重复值）

  • 循环条件：lo < hi
  • 左边界更新条件：nums[mid] < target
  • 左边界更新：lo = mid + 1
  • 右边界更新条件：nums[mid] >= target
  • 右边界更新：hi = mid 要保留mid位置上的值，即右边界不能缩小太快
  • 返回值：lo / -1

右边界二分查找

寻找有序数组中不超过目标值的最大的位置（可能有重复值）

基本和左边界相反，就是中值需要+1

  • 循环条件：lo < hi
  • 中值：mid = lo + (hi - lo) / 2 + 1 保证左边界更新能够向右移动
  • 左边界更新条件：nums[mid] <= target
  • 左边界更新：lo = mid 要保留mid位置上的值，即左边界不能缩小太快
  • 右边界更新条件：nums[mid] > target
  • 右边界更新：hi = mid - 1
  • 返回值：hi / -1

两节点最近公共祖先节点

终止条件：
当越过叶节点，则直接返回 null；
当 root 等于 p, q，则直接返回 root；
递推工作：
开启递归左子节点，返回值记为 left；
开启递归右子节点，返回值记为 right；
返回值：
left、right 同时为空 ：说明 root 的左 / 右子树中都不包含 p, q ，返回 null；
left、right 同时不为空 ：说明 p, q 分列在 root 的 异侧 ，因此 root 为最近公共祖先，返回 root ；
一个为空 ，另一个不为空 ：直接返回不为空的那一个。
